        DU_GetChildWithAttr
        		
        							
        Public Function DU_GetChildWithAttr(elParent As NotesDOMDocumentNode, targetElement As String, attrName As String, attrValue As String, flags As Long) As NotesDOMElementNode
        			        				
        Find a child DOM node with an attribute that has a particular value.
        		
        Does not recurse into the tree; looking only for immediate descendants. 
        	
        	idx	Argument	Data Type	Attributes	Comments
        1elParentNotesDOMDocumentNodenode whose children you want to search. 
        2targetElementStringByValelement name of desired child node. 
        3attrNameStringByValattribute name you want to check. 
        4attrValueStringByValattribute value of element you're looking for. 
        5flagsLongByValstring-matching flags to compare attribute, e.g. 1 for case insensitive. 
        	
         
        Code Sample
        Set titleItem = DU_GetChildWithAttr(elView, "item", "name", "$title", 1)
        		' finds the first "item" element where name='$title' (case insensitive).
