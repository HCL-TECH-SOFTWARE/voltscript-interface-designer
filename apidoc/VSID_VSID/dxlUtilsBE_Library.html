        importDXL
        		
        							
        Public Function importDXL(tgtDb As NotesDatabase, dxlfile As String, desFlg As Boolean, docFlg As Boolean, aclFlg As Boolean, signFlg As Boolean) As Boolean
        			        				
        This function is used to import an XML file containing DXL into a database. 
           
        The XML file can contain DXL for a design element, a document, or the database ACL
        	
        	idx	Argument	Data Type	Attributes	Comments
        1tgtDbNotesDatabasethe target database for the imported DXL 
        2dxlfileStringthe filepath to the file XML file containing the DXL to be imported 
        3desFlgBooleanused to set the replace flag for design elements 
        4docFlgBooleanused to set the replace flag for doc elements 
        5aclFlgBooleanused to set the replace flag for ACL elements 
        6signFlgBooleanif set to True, then the database is signed. 
           
        WARNING: This can be an expensive setting!
